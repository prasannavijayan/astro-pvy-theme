# Getting Started with Astro

Astro is a modern static site generator that helps you build fast, content-focused websites.

## Why Astro?

- **Zero JavaScript by default**: Astro ships only the HTML you need
- **Framework agnostic**: Use React, Vue, Svelte, or nothing at all
- **Content collections**: Type-safe content management
- **Islands architecture**: Only hydrate interactive components

## Getting Started

```bash
npm create astro@latest
```

Choose your preferred template and start building!
