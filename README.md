# Astro PVY Theme

A minimal portfolio + blog theme built with Astro, Tailwind CSS, MDX, and Pagefind.

## Features

- Home page with portfolio intro and featured content
- Blog with MDX support
- Projects page for portfolio
- Contact page with form
- 404 page with Pagefind search

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Tech Stack

- Astro
- Tailwind CSS
- MDX
- Pagefind (search)
